# CurrencyConverter
Приложение конвертер валют RUB, USD, EUR, GBP по Exchange Rates Data API.

## Инструменты и технологии
- Kotlin
- Retrofit
- Kotlin Coroutines
- Hilt
- Clean Architecture
- MVVM
